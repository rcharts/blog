Poirot

This blog was generated using Poirot, an elegant, blog aware, static site generator to create, manage and publish websites with reproducible content using R Markdown.

Poirot is still under development and will be available on github shortly. It is essentially a blogging layer written on top of slidify. Poirot uses markdown for text content, knitr for code chunks and mustache for templating. Poirot is highly extensible using widgets. As a publishing framework for data scientists, Poirot comes pre-equipped with support for math, visualization and tables.
